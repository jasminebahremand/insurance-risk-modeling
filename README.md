# Insurance Risk Modeling

## Overview
Developed predictive models to estimate claim likelihood and loss cost for auto insurance pricing and risk management. Combined regression and classification approaches to support underwriting and retention strategy.

## Key Findings
- LightGBM achieved R² ≈ 0.77 for loss cost prediction
- Boosting model reached ROC-AUC ≈ 0.79 for claim classification
- Insurance tenure and net premium were the strongest risk drivers
- Emerging customers showed highest loss variability, while loyal customers were lowest risk

## Tools
Python, LightGBM, XGBoost, Scikit-learn
